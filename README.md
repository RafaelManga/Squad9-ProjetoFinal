# TravelDreams ✈️🌎

## Projeto Final - SkillUp Dynatrace 2026

## Sobre o Projeto

O TravelDreams é uma aplicação web desenvolvida como projeto final do programa SkillUp Dynatrace 2026.

A plataforma simula uma agência de viagens premium, permitindo que os usuários explorem destinos turísticos, consultem pacotes de viagem, conheçam a empresa e realizem solicitações através da área de contato.

O objetivo do projeto é aplicar conceitos de desenvolvimento web e observabilidade utilizando a plataforma Dynatrace para monitoramento da experiência do usuário, performance da aplicação e análise de comportamento de navegação.

---

## Funcionalidades

- Página inicial interativa
- Navegação entre múltiplas seções
- Catálogo de destinos turísticos
- Exibição de pacotes promocionais
- Página institucional
- Formulário de contato
- Layout responsivo
- Interface moderna utilizando Glassmorphism
- Experiência de navegação otimizada
